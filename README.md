# DropSniper

DropSniper is a privacy-first, serverless price tracker built with Flutter.

## Key Features
- 100% local storage with Hive.
- Project carts with hard budget thresholds.
- Product groups that support multi-store OR tracking and automatic cheapest-link selection.
- Background scraping every 6 hours via Workmanager.
- Local notifications when a cart total drops below budget.
- Native share intake (Android/iOS) for one-tap URL tracking.
- Cyberpunk dark UI with dashboard cards, budget rings, and 30-day sparkline history.

## CI/CD (GitHub Actions)
DropSniper includes end-to-end production workflows:

- `.github/workflows/build.yml`
  - Runs `flutter analyze` + `flutter test`.
  - Builds Android release APK and Windows release EXE bundle.
  - Uploads workflow artifacts.

- `.github/workflows/release.yml`
  - Triggered by version tags (`v*`).
  - Builds signed Android release APK (when keystore secrets are configured).
  - Builds Windows release bundle and includes a code-sign placeholder step.
  - Windows release publishing is still performed even if Android build fails in the same workflow.
  - Publishes binaries to GitHub Releases with autogenerated release notes.

- `.github/workflows/version-bump.yml`
  - Manual trigger to bump `pubspec.yaml` version (`patch|minor|major`).
  - Commits updated version, creates git tag, and pushes both.

### Required release secrets
Configure these in GitHub repository secrets for signed releases:
- `ANDROID_KEYSTORE_BASE64`
- `ANDROID_KEY_ALIAS`
- `ANDROID_KEY_PASSWORD`
- `ANDROID_STORE_PASSWORD`
- `WINDOWS_CERT_BASE64` (optional for placeholder sign step)
- `WINDOWS_CERT_PASSWORD` (optional for placeholder sign step)

## Run locally
```bash
flutter pub get
flutter run -d windows
# or
flutter run -d android
```

## Build locally
```bash
flutter build apk --release
flutter build windows --release
```

## Platform notes
- Android: share target intent filter is pre-configured in `android/app/src/main/AndroidManifest.xml`.
- iOS: background and notification capabilities are configured in `ios/Runner/Info.plist`.
